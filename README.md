# GitProfilerBundle

Bundle Symfony qui expose les informations Git du dépôt courant (branche, commit court,
état *dirty*) dans le **Web Profiler**.

> 🚧 En cours de développement — pas encore publié sur Packagist.

## Prérequis

- PHP 8.3+
- Symfony 6.4 / 7.x / 8.x
